# FlowChartTool Lite

ブラウザで動くフローチャート／ブロック線図エディタです。  
ドラッグ配置・接続・マインドマップ操作・PNG/JSON出力などに対応しています。

## 使い方
1. `index.html` をブラウザで開く
2. Shapes から図形を追加
3. 図形をドラッグで配置

## 基本操作
- **移動**: 図形をドラッグ
- **複数選択**: `Ctrl` / `⌘` + クリック  
- **範囲選択**: `Select` モードで空白ドラッグ
- **パン**: `Pan` モードで空白ドラッグ
- **接続**: `Shift` 押しながら図形をクリックして線を作成
- **削除**: `Delete`

## マインドマップモード
- `Mindmap: ON/OFF` で切替
- 選択中に `Tab` で右に同じ図形を作成
- 選択中に `Enter` で下に同じ図形を作成
- 自動で線がつながります（矢印なし）

## エクスポート/インポート
- **JSON Export / Import**: Project内のボタン
- **PNG Export**: Project内のボタン  
  現在のズーム表示範囲を透過PNGで出力

## スナップ
- `Snap: ON/OFF` ボタンで切替
- スナップ間隔は数値入力で調整

## ショートカット
- `Ctrl/⌘ + C`: 選択図形をコピー
- `Ctrl/⌘ + V`: 貼り付け
- `Ctrl/⌘ + Z`: Undo
- `Ctrl/⌘ + Y`: Redo

## ファイル構成
- `index.html` UI
- `style.css` スタイル
- `app.js` ロジック
